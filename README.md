# Fralpy
A flappy bird game made using unity through following a tutorial.
Just download the zip file, unzip it and run the exe file present in there.
The game starts as soon as the file runs as there is no start ui, you just click space bar to make the bird flap and keep progressing through the pipes, to exit the game you have to do, alt+f4.
This was my first time oppening unity and I made this game by following a tutorial, I will be learning and implimenting all the features not present in this game to my future projects.
No major AI was used in building this game, it was built by followinjg a tutorial available on youtube, I followed every command given in the tutorial and built this game, I used a very small amount of AI in code completion and debugging the input system error in birdscript.cs, it was mainly built through the "tutorial", the assets were used from the tutorial, I tried just going with it as this was my first time using unity and any coding software, after this project I plan to build my own projects all by myself.
